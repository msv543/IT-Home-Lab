# Group Policy Installments

## Add GPOs: Password policies, Disable control panel, Set desktop wallpaper, Block USB storage, Map network drives, Install Chrome

## Steps Performed

1. Apply Password Policies for minimum characters 10, account lockout after 4 attempts.
2. Apply the Disable Control Panel GPO for Sales and HR OU.
3. Apply a Desktop Wallpaper to all Domain Computers.
4. Apply a "Block USB Storage" policy workstations OU that applies to Sales and HR Group.
