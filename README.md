# Active Directory Home Lab

## Overview

This repository documents my Active Directory home lab built in a virtualized Windows Server environment. The purpose of this project is to develop hands-on experience with enterprise identity management, Windows Server administration, Group Policy, and security best practices commonly used in IT Support, Systems Administration, and Cybersecurity roles.

The lab was built from the ground up and includes the deployment and configuration of core Active Directory services, user and group management, network services, security policies, and administrative delegation.

## Lab Environment

### Infrastructure
= Windows Server 2025 Domain Controller
- Windows 11 Client Workstations
- Active Directory Domain Services (AD DS)
- DNS Server
- DHCP Server
- Group Policy Management
- Microsoft LAPS
- Shared File Resources

## Technologies Used
- Active Directory Users and Computers
- Group Policy Management Console (GPMC)
- DNS Manager
- DHCP Manager
- Windows PowerShell
- Microsoft LAPS
- NTFS Permissions
- SMB File Sharing

## Labs Completed
1. Domain Controller Deployment
2. Organizational Units, Users, and Groups
3. Domain Join
4. DHCP Deployment
5. DNS Configuration
6. Group Policy, Drive Mapping, and File Sharing
7. Delegation of Control
8. Microsoft LAPS Deployment

## Purpose

This repository serves as a portfolio of practical Active Directory administration projects completed to strengthen my skills in Windows Server administration, enterprise networking, and cybersecurity.
