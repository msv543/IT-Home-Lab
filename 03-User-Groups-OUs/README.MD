# Users, Groups, and OUs 

## Objective
Create and implement Users, Groups, and OUs.

## Steps Peformed

1. Created 3 OUs in AD DS.
2. Created 1 Group for each OU.
3. Created 2 Users to assign to each OU.
4. Assigned users to their group assigned to their OU.


## Results
Created 3 OUs, 1 Group for each OU, Two users for each OU that is assigned to their group.
